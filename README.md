# coleta_de_dados_sidra
Este repositório tem como finalidade coletar dados diretamente da API do Sistema IBGE de Recuperação Automática (SIDRA). Nesse exemplo, estarei coletando dados da Produção Física Industrial Geral de Santa Catarina. Porém, o scritp pode ser adaptado para coletar outros dados da mesma fonte, desde que mudando o código da tabela.
